## Replace steps
1. Prepare your real images in PNG or JPG.
2. Keep the same file names and overwrite files under `assets/images`.
3. Hard refresh browser with `Ctrl+F5`.

## If you want JPG file names
If you rename `home-hero.png` to `home-hero.jpg`, update image paths in:
- `index.html`
- `solutions.html`
- `cases.html`
- `about.html`
- `assets/layout.js` (logo)
